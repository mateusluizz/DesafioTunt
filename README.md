# Challenge Tunts.Rocks 2024

## Installation

### Linux
1. Create a venv:
2. `python3 -m venv venv`
3. Enter in venv:
4. `source venv/bin/activate`
### Windows
1. Create a venv:
2. `python -m venv venv`
3. Enter in venv:
> #### Cmd 
> 4. `venv\Scripts\activate`  
>#### Git Bash
> 4. `. venv/Scripts/activate`

5. Install the requirements in `requirements.txt`:
6. `pip install -r requirements.txt`
7. Execute main:
> ### Windows
> 8. `python main.py`
> ### Linux
> 8. `python3 main.py`
